# Inherit

Quando usamos isso, estamos dizendo que o elemento irá ter a propriedade igual ao do elemento pai.
Ex.:

```css
div {
  color: red;
}
a {
  color: inherit;
}
```

O elemento `a` irá ter a cor vermelha
